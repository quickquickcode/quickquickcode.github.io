# quickquickcode 个人主页

🌐 **在线访问**: https://quickquickcode.github.io/

一个风格参考 [niuwh.cn](https://niuwh.cn/) 的纯静态个人主页，使用 HTML + CSS + JavaScript 构建，无第三方框架依赖。

## 特性

- 🎨 毛玻璃卡片风格，深色渐变背景
- 📱 响应式设计，适配手机、平板、桌面
- 🔗 集成 GitHub、博客、邮箱等社交链接
- ⏱️ 网站运行时间统计
- 🚀 可直接托管在 GitHub Pages

## 目录结构

```
.
├── index.html      # 主页
├── css/
│   └── style.css   # 样式文件
├── js/
│   └── main.js     # 交互脚本
└── README.md
```

## 本地预览

直接在浏览器中打开 `index.html` 即可预览。

## 修改内容

- 修改 `index.html` 中的个人信息、社交链接
- 修改 `css/style.css` 中的颜色变量（`:root`）即可快速换色
- 修改 `js/main.js` 中的 `startTime` 可调整网站起始时间

## 部署

本仓库启用 GitHub Pages 后，会自动部署到 `https://quickquickcode.github.io/`。
